### 입력 규칙 (GREEN 시 가정)

기존 `단위:값` 형식과 정합성을 위해 **`{format}:{unit}:{value}`** 접두사를 사용한다.

| 입력 예 | format | unit | value |
|---------|--------|------|-------|
| `json:meter:2.5` | json | meter | 2.5 |
| `csv:meter:2.5` | csv | meter | 2.5 |
| `text:meter:2.5` | text | meter | 2.5 |
| `meter:2.5` | text (기본값) | meter | 2.5 |

## 6. GREEN 시 구현 방향 (참고)

1. **`OutputFormatterFactory`** — 포맷 이름 → formatter 인스턴스
2. **`ConvertCommand.format` 필드** — 기본값 `"text"`
3. **`InputCommandParser`** — `{format}:{unit}:{value}` 파싱 (register 명령과 충돌 없도록 규칙 유지)
4. **`CliController`** — 요청별 formatter 선택 후 `format()` 호출
